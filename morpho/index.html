          口径与数据源说明

          关键口径（Morpho 反直觉，务必先读）
          • 协议 revenue = $0：Morpho 的 fee switch 未开启，全部借款利息归存款人 / curator，持币人零分红。$0 是真实口径，不是数据缺失。若某日 revenue≠0 = fee switch 已激活。
          • 潜在 fee switch 收入 = 年化 fees × 25%（治理上限）。理论值，当前未激活。
          • 团队年释放价值 = 未来 12 个月团队（Contributors + Founders + Strategic Partners）解锁量 × 当前币价。可变现估算，非实际到手；随币价波动。Founders 已 100% 解锁完，未来释放主要来自 Strategic Partners。
          • 坏账：已实现 vs 名义。已实现坏账 = 真实社会化损失（headline）；名义坏账含抵押品脱锚但未实现清算的部分，会被单个 exotic 死市场（如 K/USDC）灌到 $B 级，不是系统性损失。
          • APY：剔除 supply/borrow APY > 50% 的异常（脱锚）市场后，按各自资产规模加权。Morpho 有 6000+ permissionless 市场，不过滤会被极端值污染。
          • "交易量" = 净流量：借贷协议无 DEX swap volume；净借款 / 存款流量 = 借款存量 / 净 TVL 的周度差分，非 gross（借款+还款总额）。
          • 解锁 USD 按当前币价估算，价格敏感。
          数据源
          • DefiLlama — TVL / 借款 / fees / revenue / 解锁与分配（emissions datasets）
          • CoinGecko — 价格 / 市值 / 流通与总供应
          • Morpho 官方 GraphQL（blue-api.morpho.org）— 坏账 / APY / 市场 / 集中度
          • Snapshot（morpho.eth）— 治理提案
          • 更新：每周快照；数据时间见页面顶部。
          已知限制与数据严谨性
          • 多源互核：价格（CoinGecko ↔ DefiLlama，精确）、净 TVL（DefiLlama ↔ Morpho 官方，精确）、总供应量（三源）已互核；手续费为量级互核（DefiLlama ↔ Morpho 市场推算，~25% 口径近似）。坏账、解锁仍单源：坏账 Morpho 官方为链上直读权威源，无免费独立第二源（Dune 需自建 query，见验证链接）。借款因两源口径不同（Morpho totalBorrowUsd 含跨市场计价差异）未纳入互核。
          • Association 金库运营开销未完全上链，未纳入团队收益。
          • gross 交易量（借款+还款总额）未做，需交易级数据（Dune）。
          • fee switch 状态目前靠 revenue≠0 自动侦测 + Snapshot 提案追踪。

          仅供参考，非投资建议。数据可能因源口径调整而变化。
